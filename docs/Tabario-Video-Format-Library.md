# Tabario Video Format Library (Production-Ready)

## Purpose

A repeatable system of video formats designed to:

* scale content production
* maximize retention
* drive conversions
* fit Tabario’s AI-driven pipeline

Each format includes:

* Use case
* Structure (timeline)
* Visual system
* Caption style
* Editing rules
* Script template

---

# Global Editing Rules

## Pacing

* Cut every 1–2 seconds
* No dead frames

## Captions

* Always on
* Mobile-first

## Audio

* Clean voice
* Subtle music

## Visual Changes

* Every 2–4 seconds

---

# Scaling System

For each format:

* 5 hooks
* 3 caption styles
* 2 pacing variations

Total outputs per script: 30 variants

# Final Principle

Formats are assets.
Winning formats should be reused and scaled.
